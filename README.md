# Portfolio

This repository contains my personal website.

## Frontend

- Framework: Next.js
- Styling: Tailwind CSS
- Sections:
  - Blog
  - Work
  - About Me
  - Admin
- Deployment: Vercel

## Backend

- Language: Go
- Frameworks: Gin and Gorm
- Functionality:
  - Simple CRUD operations
  - Authentication using JWT
  - Token stored in a cookie
- Deployment: Fly.io
- Database: PostgreSQL
